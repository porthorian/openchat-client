# OpenChat Client v0.0.7

## Highlights
- Added mentions support in chat with mention-pill rendering and current-user mention highlighting.
- Expanded channel/category management with create channel plus create, rename, reorder, and delete category flows.
- Added audio/video media test options in user settings.
- Improved voice, video, and screen share negotiation behavior, including STUN/TURN-backed signaling paths and codec handling.
- Updated the default server target to `chat.pennylabs.com`.

## Fixes
- Fixed OpenGraph rendering behavior to avoid double escaping/unescaping issues.
- Fixed a temporary identity randomness issue.
- Fixed release/CI stability issues that previously caused excessive asset uploads and failing CI checks.

## Reliability & Dependency Updates
- Refreshed runtime and tooling dependencies across the release window to improve reliability and security posture.
