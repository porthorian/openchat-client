# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records for the OpenChat Client.

## ADR Process
1. Copy `ADR_TEMPLATE.md` to a new numbered file.
2. Use a short, descriptive slug in the filename.
3. Set status (`Proposed`, `Accepted`, `Superseded`, `Deprecated`).
4. Link related issues, pull requests, and feature specs.
5. Update this index.

## Naming Convention
- Format: `NNNN-short-title.md`
- Example: `0005-renderer-routing-strategy.md`
