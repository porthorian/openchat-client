# Support

## Where to Ask for Help
- Use GitHub Discussions for usage and architecture questions.
- Use GitHub Issues for reproducible bugs and concrete feature requests.

## Before Opening a Request
- Check existing issues and discussions.
- Include environment details and exact reproduction steps.
- Attach screenshots or recordings for UI behavior problems.

## Security Issues
Do not post vulnerabilities publicly. Follow `SECURITY.md`.

## Scope Reminder
This repository is client-only. Backend implementation support requests should be directed to the backend repository once published.
